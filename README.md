# Soltal Pet Market

Starter de tienda online hecha con código usando Next.js.

## Incluye

- Página principal
- Productos con imágenes reales
- Categorías y buscador
- Detalle de producto
- Carrito
- Checkout
- Factura descargable
- API inicial para pedidos
- Panel admin inicial
- SQL para Supabase

## Ejecutar

```bash
npm install
npm run dev
```

Página:

```txt
http://localhost:3000
```

Admin:

```txt
http://localhost:3000/admin
```
